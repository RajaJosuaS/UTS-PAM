### Cara menginstall aplikasi (dalam mode pengembangan)

1. clone repository
```
git clone https://github.com/RajaJosuaS/UTS-PAM
```

2. pindah direktory
```
cd UTS-PAM
cd Contact
```

3. install dependensi yang diperlukan
```
npm install
```
4. install expo
```
npm start
```

5. build to apk (android)
```sh 
eas build:android
```
